# IoT Temperature and Humidity Monitoring System

## Project Overview
This project is an IoT based Temperature and Humidity Monitoring System developed using ESP32, DHT22 sensor, MQTT protocol and Node-RED dashboard.

The system monitors environmental conditions in real time and displays live data through gauges, graphs and alert notifications.

---

## Technologies Used
- ESP32
- DHT22 Sensor
- MQTT Protocol
- Node-RED
- Wokwi Simulator

---

## Features
- Real-time temperature monitoring
- Real-time humidity monitoring
- MQTT communication
- Live dashboard visualization
- Alert notification system
- Graphical data representation
